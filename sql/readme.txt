
[1] Download scouts.zip and deploy into xampp/htdocs/scouts

[3] Go to SQL Admin panel (phpmyadmin); Create a database called "scouts".

[4] Import the SQL file (xampp/htdocs/scouts/sql/girl_scouts.sql) into scouts database.

[5] Launch the quiz_master application
http://localhost/scouts
